# **App Name**: ApexBuild

## Core Features:

- Homepage Showcase: Display company information, expertise, and a curated selection of projects.
- Project Portfolio: Display a comprehensive list of all completed projects with filtering and search options.
- Case Study Reports: Present in-depth reports of selected projects with detailed analysis, progress tracking, photos, and videos.
- Project Submission Form: Allow users to submit project details via an interactive form.
- Contact Information: Display contact numbers, board member profiles, social media links, and a Google Maps schematic of the company's address.
- User Authentication: Secure user registration and login functionality to manage project submissions and access project information.
- AI-Powered Project Evaluation: Employ an AI tool that assesses submitted project details to determine preliminary feasibility and provide estimated resource requirements, assisting project managers in prioritizing and planning new projects.

## Style Guidelines:

- Primary color: Strong blue (#3498DB) to evoke trust, stability, and professionalism in the construction business.
- Background color: Light gray (#ECF0F1), nearly desaturated from the primary, provides a clean and neutral backdrop.
- Accent color: Soft violet (#9B59B6), is analogous to the primary blue and will highlight important CTAs and details, drawing attention in a subtle yet effective manner.
- Headline font: 'Playfair', serif, for elegant and professional titles; body font: 'PT Sans', sans-serif, for readable and clear paragraphs.
- Use clean, professional icons related to construction, architecture, and project management.
- Maintain a clean and structured layout to showcase projects effectively. Use clear sections for company info, projects, and contact details.
- Implement subtle animations for transitions and loading states to improve user engagement.
